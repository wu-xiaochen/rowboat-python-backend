# Contributing to Rowboat Python Backend

感谢您对 Rowboat Python Backend 项目的关注！

## 开发指南

### 代码规范

- 使用 Python 3.10+
- 遵循 PEP 8 代码风格
- 使用类型提示（Type Hints）
- 编写清晰的文档字符串

### 提交规范

- 使用清晰的提交信息
- 一个提交只包含一个逻辑变更
- 提交前运行测试确保通过

### 测试

- 为新功能添加测试
- 确保所有测试通过
- 保持测试覆盖率

## 报告问题

如果发现问题，请提供：

- 详细的错误描述
- 重现步骤
- 环境信息
- 相关日志

## 功能请求

欢迎提出新功能建议，请说明：

- 功能用途
- 使用场景
- 预期效果
